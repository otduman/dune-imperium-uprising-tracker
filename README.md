# Dune Imperium Uprising Tracker

Welcome to the **Dune Imperium Uprising Tracker**! 

This is a web app built to easily keep track of your board game sessions. It features a heavy cinematic, sci-fi design that matches the look and feel of the board game.

## ✨ Features

- **Track Matches:** Log the game date, players, their final scores, and the Leader they played.
- **Tiebreakers:** Automatically resolves ties! It uses the official game rules (Spice → Solari → Water → Garrison) and displays real token graphics (the ones from the game!).
- **Leaderboard:** View a standings table to see who has the most wins, best average score, and longest winning streaks.
- **Player Profiles:** Click on a player to see their detailed stats, favorite Leaders, and biggest wins.
- **Match History:** See an endless feed of all games played.
- **Official Assets:** Uses actual high-quality card graphics and tokens pulled directly from Dune Cards Hub.

## 🚀 How to Run It Locally

To start the app on your own computer, make sure you have Node.js installed. Then, open a terminal in this folder and type:

1. **Install dependencies:**  
   `npm install`

2. **Start the app:**  
   `npm run dev`

3. **Open it:**  
   Open [http://localhost:3000](http://localhost:3000) in your web browser.

## 🛠️ Built With

- **Next.js & React** – The core framework running the website.
- **Tailwind CSS** – Used for all the styling, colors, and layouts.
- **Lucide Icons** – For clean, scalable UI icons.
- **Lucide React** – Used for smaller SVG interface icons.
